Remark 1.5.
Um sistema linear é dito compatível se admite soluções. Sistemas compatíveis podem ter solução única ou infinitas soluções (Não é possível que um sistema linear tenha \(n\) soluções \(n=2, 3, 4, \ldots \) soluções). Um sistema linear é dito incompatível se não admite soluções.
