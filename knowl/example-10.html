O sistema

\begin{align*} x - 2y - 3z\amp= 1,\\ 2x -2y - 2z \amp= 6,\\ 2x - 4z \amp= 8,\\ 5x -7y + 4z \amp= 46/3, \end{align*}
tem matriz ampliada
\begin{equation*} \left(\begin{array}{ccc|c} 1 \amp -2 \amp -3 \amp 1\\ 2 \amp -2 \amp -2\amp 6\\ 2 \amp 0 \amp -4 \amp 8\\ 5 \amp -7 \amp 4 \amp 46/3 \end{array} \right) \sim \cdots \sim \left(\begin{array}{ccc|c} 1 \amp 0 \amp 0 \amp 14/3\\ 0 \amp 1 \amp 0\amp 4/3\\ 0 \amp 0 \amp 1 \amp 1/3\\ 0 \amp 0 \amp 0 \amp 0 \end{array} \right) \end{equation*}
assim o sistema tem solução única \(x=14/3,\,\, y= 4/3, \,\, z = 1/3\text{.}\)
