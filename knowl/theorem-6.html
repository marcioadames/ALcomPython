Theorem 2.44. Propriedades do determinante.

Seja \(A\) uma matriz \(n \times n\text{:}\)

  1. Se trocarmos duas linhas de \(A\) de lugar, o determinante troca de sinal, ou seja: \(\det(T_{ij}A) = - \det(A)\text{.}\)

  2. Se multiplicamos uma linha de \(A\) por uma constante não nula, o determinante é multiplicado por \(\alpha\text{,}\) ou seja: \(\det(D_{i}(\alpha)A) = \alpha \det(A)\text{.}\)

  3. Se somamos a uma linha de \(A\) outra linha multiplicada por uma constante, o determinante não se altera, ou seja: \(\det(U_{ij}(\alpha)A) = \det(A)\text{.}\)

  4. \(\det(A) = \det(A^T)\text{.}\)

  5. Se \(A\) tem duas linhas ou duas colunas iguais, \(\det(A) = 0\text{.}\)

  6. Se \(A\) tem uma linha nula ou uma coluna nula, \(\det(A) = 0\text{.}\)

  7. Se \(B\) é uma matriz \(n \times n\text{,}\) \(\det(AB) = \det(A)\det(B)\text{.}\)
