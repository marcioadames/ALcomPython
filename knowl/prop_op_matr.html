Theorem 2.9. Propriedades das operações básicas.

Dados \(\alpha, \beta \in \mathbb{R}\) e matrizes \(A, B\) e \(C\text{.}\) Então, se as operaçõesa seguir estiverem definidas (as dimensões das matrizes forem adequadas para permitir as operações), vale:

  1. \(\displaystyle A + B = B + A, \)

  2. \(\displaystyle (A + B) + C = A + (B + C), \)

  3. \(\displaystyle (AB)C = A(BC), \)

  4. \(\displaystyle A(B+C) = AB + AC, \)

  5. \(\displaystyle (A+B)C = AC + BC, \)

  6. \(\displaystyle (\alpha \beta) A = \alpha(\beta A), \)

  7. \(\displaystyle \alpha(AB) = (\alpha A)B = A(\alpha B), \)

  8. \(\displaystyle (\alpha + \beta) A = \alpha A + \beta A. \)
