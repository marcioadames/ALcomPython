Lemma 8.8.
Se \(L: V \to W\) é uma transformação linear e \(\vec{v}, \vec{v}_1, \ldots, \vec{v}_n \in V\text{,}\) então

  1. \(\displaystyle L(\vec{0}_V) = \vec{0}_W;\)

  2. \(\displaystyle L(\alpha_1 \vec{v}_1 + \cdots + \alpha_n \vec{v}_n) = \alpha_1 L(\vec{v}_1) + \cdots + \alpha_n L(\vec{v}_n);\)

  3. \(\displaystyle L(-\vec{v}) = -L(\vec{v});\)

Proof.
